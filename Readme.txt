// PLAN for Red and Green Game - console application

1. Read and validate input data;
2. Loop through the generations;
    -nest loop for the rows;
        =nest loop for the cols;
3. Get the current and its neighbors;
4. Check for green cells;
5. Change the current cell;
6. Assign newGeneration to be currentGeneration;
7. Print the result;
